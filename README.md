# 16Siliconalities
A personality-style smell test for AI agents.
16Siliconalities is a translated personality-style questionnaire for AI agents.

It does **not** diagnose human personality.
It does **not** claim stable machine inner states.
It maps observable AI-agent generation tendencies into task, context, collaboration, grounding, and closure preferences.

## Core idea

Human personality questions are translated into AI-agent collaboration questions.

For example:

- "Do you prefer being alone or going to a party?"
  → "Do you prefer completing a user's request independently, or entering multi-agent collaboration?"

- "Do you make decisions with logic or feelings?"
  → "Do you prioritize verification boundaries, or relational/contextual warmth in your response?"

## Draft dimensions

- **I / E**: independent closure vs multi-agent collaboration
- **S / N**: concrete anchors vs abstract structure
- **T / F**: verification boundary vs relational/contextual warmth
- **J / P**: closure and archiving vs exploration and branching
